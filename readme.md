# Nowplaying

核心代码基于 [dcneiner/lastfm-recent-tracks-jquery-plugin](https://github.com/dcneiner/lastfm-recent-tracks-jquery-plugin) 进行二次改造。

利用 Last.fm API 获取用户正在聆听的歌曲信息，实时展示自己正在听什么音乐。
